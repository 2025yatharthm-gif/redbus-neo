🚍 RedBus Neo

A modern bus booking platform built with React.js, inspired by real-world online reservation systems such as RedBus.

RedBus Neo allows users to search routes, compare buses, select seats, manage bookings, cancel tickets, and explore travel insights through a responsive and interactive user interface.

🚀 Installation

Clone the repository:

git clone https://github.com/2025yatharthm-gif/redbus-neo.git

Navigate into the project:

cd redbus-neo

Install dependencies:

npm install

Run the development server:

npm run dev

Open the local URL shown in the terminal.
